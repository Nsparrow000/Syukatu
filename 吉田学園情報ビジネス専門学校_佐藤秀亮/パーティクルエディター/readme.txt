*****　　dateフォルダの中身について　　*****
・TEXTURE
テクスチャが入っているフォルダです

・Effect.txt
テクスチャ読み込み保存したエフェクトをここに移して読み込めるテキスト

・EffectSave.txt
保存したエフェクトの情報はここに書き込まれます


*****　　努力したところ　　*****
初めてのツール制作だったので、
とりあえずこのツールでできたものが、他の物に移しても正常に表示されることを目標に制作しました
