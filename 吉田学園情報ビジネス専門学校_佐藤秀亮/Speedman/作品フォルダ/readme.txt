３Dアクションゲーム「speedman.com」　readme

・開発人数
5人
※主にメニュー画面や、エフェクト等のアウトゲーム部分を担当

・開発環境
Visual Studio 2015

・使用言語
C++

・製作期間
約２週間

*****　　注意点　　*****
・ESCキーについて
タイトル画面で押すとexeの終了
それ以外の画面だとメニュー又は無反応になります

・コントローラーについて
キーボードとコントローラー両方に対応していますが（コントローラー推奨）
コントローラー使用の際はexe起動する前に差してください

*****　　dateフォルダの中身について　　*****
・BGMフォルダ
BGMが入っているフォルダです

・MODEL
3Dモデルが入っているフォルダです、Playerフォルダにはプレイヤーのモデル一式が入ってます

・SEフォルダ
SEが入っているフォルダです

・Textureフォルダ
Textureが入っているフォルダです

・3DTexNameRead.txt
3Dの部分で使うテクスチャのパスを読み込むファイルです

・chara.txt
キャラクターのモーションデータ、モデルのパスが入っているファイルです

・EffectTexNameRead.txt
エフェクトのテクスチャのパスが書かれてるフォルダです

・Rank.txt
ランキングのデータです

・stage.txt
ステージのデータです

・TexNameRead.txt
2Dの部分で使うテクスチャのパスを読み込むファイルです

・tutorial.txt
チュートリアルステージのデータです
