2D・3D両用エフェクトツール　　

制作者：佐藤秀亮　syusuke sato

・開発環境
Visual Studio 2015

・使用言語
C++

・基礎部分の製作期間
約2ヶ月

・実行手順
エフェクトツール/作品フォルダ/実行ファイル/Effect_Tool.exeを起動してください

*****　　dateフォルダの中身について　　*****
・MODELフォルダ
3DModeにて使用するxファイル等が入っています

・Textureフォルダ
exe内で使う、テクスチャーが2D、3Dに分けられて入っています

・3DMode.txt
exe内の3DModeで配置するモデル、操作キャラの情報などが書かれています

・EffectMesh.txt
軌跡用のエフェクトを作成する際のテクスチャを読み込みます

・Motion.txt
操作キャラに関したすべての情報が入っています
（モーション、パーツ、ジャンプ力、移動量etc）

・SaveText.txt
エフェクトの情報を記録した際記録した情報が出力されるテキストです
※保存した後に再度保存するとこのテキストは上書き保存されるので
　保存した際は別のテキスト等に張り付けて利用してください

・Tex2DNameRead.txt
2D用のエフェクト等のテクスチャ情報が入っているテキストです

・Tex3DNameRead.txt
3D用のエフェクト等のテクスチャ情報が入っているテキストです

・IntermeidiateText2D.txt
中間セーブ情報が入っているテキストです(2D用)

・IntermeidiateText3D.txt
中間セーブ情報が入っているテキストです(3D用)

・Preset.txt
3Dプリセット（複数同時再生）用のテキスト

・EffectData.txt
エフェクト情報のテキスト

・2DSetTex.txt
2Dモードの操作ボタンを配置するテキスト

・3DSetTex.txt
3Dモードの操作ボタンを配置するテキスト

*****　　努力した点　　*****
このツールを使って2D、3Dのエフェクトを制作できるように

できるだけいろいろな動きができるように、極座標系を使った球体のエフェクト、
ベジェ曲線を使った曲がるエフェクトを作成できるようにしたところ
